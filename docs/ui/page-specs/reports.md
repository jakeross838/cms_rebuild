- Revenue: Signed contracts + approved change orders

  - Materials: Cost codes with category = "Material"
  - Subcontractors: Cost codes with category = "Subcontractor"
  - Equipment: Cost codes with category = "Equipment"

- Period selection: MTD, QTD, YTD, Custom
- Compare to prior period/year
- Trend analysis
- Export for accounting
