# Module 7: Scheduling

**Phase:** 2 - Construction Core
**Status:** TODO

## Description
Project scheduling with Gantt charts, task dependencies, and resource assignment. Enables builders to plan construction phases, track progress, and identify delays.

## Key Features
- Gantt chart with drag-and-drop editing
- Task dependencies (FS, FF, SS, SF)
- Milestone tracking
- Resource/crew assignment to tasks
- Critical path calculation
- Baseline vs. actual comparison
- Schedule templates for common project types
- Calendar view and timeline view
