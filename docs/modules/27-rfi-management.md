# Module 27: RFI Management

**Phase:** 4 - Intelligence
**Status:** TODO

## Description
Request for Information (RFI) creation, routing, and tracking. Manages the question-and-answer process between field, office, architects, and engineers with full audit trail.

## Key Features
- RFI creation with rich text and attachments
- Routing to appropriate parties (architect, engineer, owner)
- Response tracking with due dates
- Status workflow (Open, Pending, Answered, Closed)
- Cost and schedule impact assessment
- RFI log with filtering and search
- Drawing/spec reference linking
- Overdue RFI alerts
