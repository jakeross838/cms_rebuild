# Module 20: Estimating Engine with Intelligence

**Phase:** 4 - Intelligence
**Status:** TODO

## Description
Full estimating system with assemblies, unit costs, and AI-assisted pricing. Enables builders to create accurate estimates that convert into project budgets and proposals.

## Key Features
- Estimate creation with hierarchical line items
- Assembly/recipe library (pre-built cost groups)
- Unit cost database with regional adjustments
- AI-suggested pricing based on historical data
- Estimate versioning and comparison
- Markup and margin configuration
- Estimate-to-budget conversion
- Proposal generation from estimates
