# Module 16: QuickBooks Integration

**Phase:** 3 - Financial Power
**Status:** TODO

## Description
Bidirectional sync with QuickBooks Online for accounting data. Syncs vendors, invoices, payments, and chart of accounts to eliminate double-entry between construction and accounting systems.

## Key Features
- OAuth2 connection to QuickBooks Online
- Vendor/supplier sync (bidirectional)
- Invoice push to QuickBooks
- Payment status sync from QuickBooks
- Chart of accounts / cost code mapping
- Sync status dashboard and error handling
- Manual and scheduled sync options
- Conflict resolution for bidirectional updates
