# Module 10: Contact/Vendor Management

**Phase:** 2 - Construction Core
**Status:** TODO

## Description
Centralized directory of subcontractors, suppliers, clients, and other contacts. Tracks company info, insurance status, trade specialties, and contact history.

## Key Features
- Contact and company directory
- Trade/specialty categorization
- Insurance certificate tracking with expiration alerts
- Contact tagging and grouping
- Communication history log
- Favorite/preferred vendor lists
- Import contacts from CSV/vCard
- Contact sharing across projects
