# Module 11: Basic Invoicing

**Phase:** 2 - Construction Core
**Status:** TODO

## Description
Core invoice management for receiving and tracking vendor/subcontractor invoices. Supports manual entry, status tracking, and approval workflows before payments.

## Key Features
- Invoice creation and data entry
- Invoice status workflow (Draft, Submitted, Approved, Paid)
- Cost code and project allocation
- Approval routing based on amount thresholds
- Invoice line item management
- Due date tracking and aging reports
- Basic duplicate detection
- Invoice attachment storage
