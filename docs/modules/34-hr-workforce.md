# Module 34: HR & Workforce

**Phase:** 5 - Full Platform
**Status:** TODO

## Description
Employee and workforce management including time tracking, crew assignments, certifications, and basic HR functions tailored for construction companies.

## Key Features
- Employee directory and profiles
- Time tracking with geofenced clock-in/out
- Crew assignment to projects
- Certification and license tracking with expiration alerts
- PTO and absence management
- Payroll data export
- Labor cost reporting by project
- Employee onboarding checklists

## Related Gap Items
- Gap items: G-295 through G-303

## Dependencies
- Module 1: Auth & Access (employee accounts)
- Module 3: Core Data Model (project assignments)
- Module 7: Scheduling (crew scheduling)
- Module 5: Notification Engine (certification alerts)

## Notes
- Larger builders need this; smaller builders may use external HR tools.
