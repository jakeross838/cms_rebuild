# Module 8: Daily Logs

**Phase:** 2 - Construction Core
**Status:** TODO

## Description
Digital daily log entries capturing weather, crew hours, work performed, materials used, and site conditions. Replaces paper-based field reporting with structured, searchable records.

## Key Features
- Daily log creation with date and project selection
- Weather conditions capture (manual or auto-fetch)
- Crew/labor hours tracking
- Work performed narrative entries
- Material deliveries and usage logging
- Site visitor log
- Photo attachments per log entry
- Supervisor sign-off workflow
