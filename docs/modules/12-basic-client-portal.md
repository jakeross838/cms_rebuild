# Module 12: Basic Client Portal

**Phase:** 2 - Construction Core
**Status:** TODO

## Description
A limited client-facing view allowing homeowners/clients to see project progress, photos, and basic updates. Provides transparency without exposing internal financial or operational details.

## Key Features
- Client login with restricted access
- Project progress overview
- Photo gallery per project
- Basic messaging between client and builder
- Selection status visibility
- Document sharing (plans, contracts)
- Project timeline/milestone view
