# Module 15: Draw Request Generation

**Phase:** 3 - Financial Power
**Status:** TODO

## Description
Generates AIA-style draw requests (G702/G703) and custom draw formats for submitting payment applications to lenders or owners. Tracks approval status and payment receipt.

## Key Features
- AIA G702/G703 form generation
- Custom draw request formats
- Schedule of values management
- Percent complete tracking per line item
- Retainage calculation and tracking
- Supporting document assembly (lien waivers, invoices)
- Draw request approval workflow
- Historical draw tracking and reporting
